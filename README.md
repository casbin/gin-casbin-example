# gin-casbin-example
[gin-casbin-example](https://github.com/casbin/gin-casbin-example) is intended to show how to use [Casbin](https://github.com/casbin/casbin) in [Gin](https://github.com/gin-gonic/gin).

##### To run the test
```bash
go test -v
```
